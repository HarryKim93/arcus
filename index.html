아커스 (Arcus)는 memcached 1.4 버전을 기반으로 네이버 (NAVER) 서비스들의 요구 사항에 맞게 개발한 메모리 캐시 클라우드 입니다.

아커스는 백엔드 저장소인 데이터베이스의 앞단에 위치하여 hot-spot 성격의 데이터를 캐싱하여 서비스함으로써, 여러 서비스 응용들에게 빠른 응답성을 제공하면서 데이터베이스로의 부하를 경감시켜 줍니다. 그 외에도 복잡한 계산에 의한 결과물 또는 웹 처리상의 중간 데이터 등을 신속히 저장하거나 조회할 수 있으며 캐싱된 데이터를 여러 프로세스들 간에 공유하여 사용할 수 있습니다.

아커스는 주키퍼 (ZooKeeper)를 통해 여러 개의 캐시 서버를 묶은 클러스터를 형성하여 관리할 수 있습니다. 또한 기본적인 Key-Value 데이터 모델 외에 여러 개의 값을 구조화된 형태로 저장하고 조회하게 하는 컬렉션 자료구조 (List, Set, B+Tree)를 제공하고 있습니다.

이 아커스가 곧 오픈소스로 여러분 곁에 다가갑니다. 기대해 주세요!
